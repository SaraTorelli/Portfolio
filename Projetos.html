Declaração

Vem ver!
Este projeto foi inteiramente desenvolvido utilizando HTML e CSS para criar uma página especial dedicada ao meu namorado. A ideia por trás da página é me declarar de uma forma criativa e envolvente, com elementos que tornam a experiência única. A página inclui um carrossel de fotos, onde são exibidas imagens que representam momentos especiais do nosso relacionamento. Também há uma seção de texto, que traz uma mensagem de carinho e sentimentos. O fundo da página foi projetado com animação de estrelas, criando um ambiente mágico e romântico. Além disso, há um contador que exibe o tempo exato de quanto tempo estamos juntos, marcando cada dia dessa linda jornada. Para tornar a experiência ainda mais envolvente, uma música foi inserida na página, proporcionando uma trilha sonora que acompanha a mensagem de amor e afeto. Tudo foi pensado para criar uma experiência única e emocional, utilizando recursos simples de HTML e CSS para expressar um sentimento profundo de forma criativa e inesquecível.
