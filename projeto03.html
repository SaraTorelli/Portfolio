Protótipo Loja de Roupas

Aplicação
Este projeto foi desenvolvido inteiramente em Kotlin, uma linguagem moderna e eficiente, escolhida pela sua ótima compatibilidade com o Android. A ideia por trás dele é criar um teste para um futuro sistema de loja online, que estou prestes a finalizar para um freelancer. Mesmo sendo um projeto de teste, ele já conta com várias funcionalidades interessantes que servem como base para o que está por vir. Logo de início, o aplicativo abre com uma splash screen, uma tela de boas-vindas que aparece antes do conteúdo principal. Além disso, outras telas explicativas são mostradas em sequência, com um toque de animação que imita o movimento de água quando o usuário interage, dando uma sensação leve e agradável ao app. Depois que essas telas introdutórias terminam de carregar, o usuário é levado a uma aba de login, que surge de maneira suave, com uma animação sutil. Caso a pessoa ainda não tenha uma conta, pode ir para a aba de cadastro e registrar seus dados. Quando o login é feito, o aplicativo mostra uma aba de produtos, que será preenchida com as opções da loja futuramente. Tudo foi pensado para oferecer uma experiência simples e intuitiva, criando uma navegação fluida e envolvente.
